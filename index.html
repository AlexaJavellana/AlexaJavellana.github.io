work at ibm
Using IBM technologies including Bluemix, Watson, I spent the summer of 2018 as an integrative software developer for the Tax Portal team under IBM's CIO. As a developer on the Portal team, my main goal was to create new features and integrate new services into a pre-existing platform called the Tax Information Portal. This included implementing a NPS into the platform, hooking up security services such as Contrast to improve response time to threats or bugs, and creating an interactive dashboard visualizing Portal user data activity with Python scrapes and Bokeh graphics from New Relic statistics, formatted with Pandas and Numpy.
